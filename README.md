# Automação de sistema contábil

Este projeto automatiza o lançamento de transações financeiras em um sistema de contabilidade a partir de uma planilha Excel.

## Arquivos incluídos
- automacao.py: script principal de automação com Playwright.
- lancamentos.xlsx: planilha com os lançamentos a serem importados.

## Dependências
Instale as dependências com:

```bash
pip install -r requirements.txt
```

## Como executar
```bash
python automacao.py
```
